\## 📌 项目简介



本项目旨在对贵州大学“感恩书屋”所藏的陈祖武先生赠送图书、题跋及珍贵信件进行数字化整理与深度挖掘。通过 Python 自动化数据清洗，提取图书背后的隐性信息，并利用现代前端技术构建交互式的学术交流可视化大屏。



该项目不仅是一项特藏文献的建档工作，更是一次结合历史文献学与数字人文（Digital Humanities）的跨学科实践，直观还原了陈祖武先生跨越半个世纪的学术网络与代际传承。

\## 📂 项目文件结构与说明



本项目保留了从原始数据清洗到最终可视化的完整数字流水线：



\### 💻 核心代码

\* `data\_analysis.ipynb`：核心数据处理 Jupyter 源码。包含正则时间提取、学者实体识别、合捐拆分算法、图表生成逻辑等全套数据清洗与可视化脚本。



\### 📊 数据流水线 (Data Pipeline)

\* `陈祖武先生捐赠书籍整理书目.xlsx`：原始文献总目录基准表。

\* `有附件的书目筛选结果.xlsx`：针对带有信件、题跋等附件的文献专项提取表。

\* `感恩书屋有附件的书目分类.xlsx`：学者关系（师长、友朋、晚辈等）分类与核对表。

\* `赠阅图书整理+时间版本.xlsx`：时间提取过程的中间数据备份。

\* `赠阅图书整理\_时间已提取.xlsx`：\*\*核心底层数据\*\*，经过深度清洗并提取出规范化时间，直接驱动前端大屏。



\### 🏆 统计分析成果 (Analytics)

\* `赠书人去重总名单.xlsx`：通过算法拆分合捐情况后，导出的所有独立赠书学者名录。

\* `陈祖武先生赠书人排行榜.xlsx`：以赠书册数为依据生成的高频赠书学者/机构数据榜单。



\### 🌐 可视化交互大屏 (Visualizations)

\* `陈祖武先生受赠图书明细.html`：轻量版前端交互检索大屏（包含 ECharts 历年受赠趋势与 DataTables 检索底座）。

\* `陈祖武先生受赠图书明细\_含附件内容.html`：\*\*完整版大屏\*\*。在轻量版基础上增设了“附件内容”展示列，可直接检索题跋、信件等一手文献原文。

\* `陈祖武赠书关系图谱.html`：基于 Pyvis 构建的物理网络拓扑图。以陈祖武先生为核心，按代际关系将学者映射于同心圆轨道，节点大小直观反映赠阅数量。



\### 📝 项目文档 (Docs)

\* `陈祖武赠书整理.docx`：特藏文献整理工作报告与学术脉络总结。

\* `需要的工作.docx`：项目初期的任务清单与需求规划。
